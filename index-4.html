The transition-timing-function property can have the following values: (1) ease - specifies a transition effect with a slow start, then fast, then end slowly (this is default) (2) linear - specifies a transition effect with the same speed from start to end (3) ease-in - specifies a transition effect with a slow start (4) ease-out - specifies a transition effect with a slow end (5) ease-in-out - specifies a transition effect with a slow start and end

What are CSS Animations?

An animation lets an element gradually change from one style to another. You can change as many CSS properties you want, as many times as you want. To use CSS animation, you must first specify some keyframes for the animation. Keyframes hold what styles the element will have at certain times.

When you specify CSS styles inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain times
